Summary

webwigo is an online emulator for Wherigo cartridges. It primary targets authors of such cartridges, allowing them to debug their code at home before going to the field and finalize the very last details!

Browser

webwigo relies on Modernizr to check if the required features (typedarray, xhr2, cors and webworkers) are enabled. However, the application is built to work in the latest desktop browsers. As such, Internet Explorer 10, Firefox 24 and Chrome 30 are supported.

Server

webwigo runs entirely in the browser. There are no data exchange with the server. The privacy of your source code is guaranted.

Load

The emulator currently accepts 3 formats as input: .lua (source code, without medias), .gwz (zip archive of source code, with all medias included) and .gwc (compiled version of a .gwz file). The running cartridge is stopped only if the new one is loaded successfully.

.lua

The provided file should contain a valid Wherigo source code. webwigo extracts the name of the cartridge from this file. An error is thrown if this name is not found. If nothing happens after clicking the "Run" button, check the "Console" tab to find out the issue (syntax error, etc.)

.gwz

The provided file must be a valid zip archive. It must contain a valid Wherigo source code (as specified above) and zero or more medias (images and/or sounds). Depending on the number of files, loading may take some time.

.gwc

The provided file must be a valid compiled Wherigo cartridge. The "Medias" tab is disabled and the completion code is never displayed (small protection against cheating).

Reload

The current loaded cartridge (.lua, .gwz or .gwc) can be refreshed at any time by clicking the "Reload" button (the running cartridge is stopped).

Run

Clicking the "Run" button starts (or restarts) the cartridge. Depending on your browser, on your computer speed and on the cartridge, the initialization may take some time. Since it's not possible to show a progress bar in that case, make sure to be patient, until the "Splash Screen" is displayed!

Map

The "Map" tab displays the cartridge starting location (red marker, fixed), the player location (green marker, draggable), the GPS coordinates, the GPS accuracy (green, orange or red) and the active zones (green for visible, red for not visible).
